# Pokemon-Slick2D
This is a pokemon red version clone made with Slick-2D. It's far from being finished (Doesn't even include pokemon), this project has been made for fun for a RPG creating contest.

# Controls
* [Z]: A button
* [A]: B button
* [Arrows]: Move
* [Enter]: Start menu
